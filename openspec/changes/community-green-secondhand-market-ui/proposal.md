## Why

The current secondhand marketplace opportunity is to make local, face-to-face trade feel safer and more intentional. A community-first experience can reduce friction by emphasizing proximity, trust, and sustainable reuse instead of anonymous shipping-first behavior.

## What Changes

- Introduce a Vue-based marketplace frontend with a calm, rounded visual language in a "community green" palette.
- Prioritize nearby listings and face-to-face trading flows over long-distance fulfillment.
- Add map-based exploration for discovering items and sellers in the user's area.
- Add seller trust scoring to surface reliability signals before contact.
- Add real-time messaging so buyers and sellers can coordinate quickly.
- Add meetup-point suggestions to help users choose practical handoff locations.

## Capabilities

### New Capabilities
- `community-secondhand-market-ui`: Vue front-end shell, navigation, listing discovery, and the overall community-first marketplace experience.
- `map-exploration`: Interactive geographic browsing for nearby listings, seller clusters, and local inventory discovery.
- `seller-trust-score`: Trust and reputation signals that help users assess sellers before initiating a trade.
- `real-time-chat`: In-app messaging between buyers and sellers for negotiation and coordination.
- `meetup-point-suggestions`: Suggested public meet-up locations based on proximity, convenience, and trade safety.

### Modified Capabilities

None.

## Impact

- New Vue frontend screens, shared UI components, and marketplace interaction states.
- Location-aware browsing data and geospatial presentation logic.
- Messaging UI and supporting real-time communication integration points.
- Trust and meetup recommendation presentation surfaces.
- Design system updates for the community green palette and rounded visual style.
