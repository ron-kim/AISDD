## ADDED Requirements

### Requirement: Community-first marketplace home
The UI SHALL present a Vue-based home experience that emphasizes nearby, face-to-face secondhand trades and uses a community green, rounded visual style.

#### Scenario: First visit to home
- **WHEN** the user opens the marketplace
- **THEN** the UI SHALL show nearby-first discovery content, primary actions for browsing and messaging, and the community-green rounded theme

### Requirement: Responsive discovery layout
The UI SHALL render usable layouts on mobile and desktop without losing access to browse, map, trust, and chat entry points.

#### Scenario: Small screen layout
- **WHEN** the viewport is narrow
- **THEN** the navigation and primary actions SHALL remain accessible without horizontal scrolling
